# Random Task Generator

**Автор:** Иван Мироненко

## Описание
Приложение для генерации случайных задач из предопределённого списка. Поддерживает фильтрацию по типу задачи, добавление новых задач и сохранение истории в JSON.

## Возможности
- Генерация случайной задачи
- Фильтрация по типу: учёба, спорт, работа
- Добавление новых задач
- Сохранение и загрузка истории задач

## Примеры использования
1. Запустите `main.py`.
2. Нажмите "Сгенерировать задачу" для получения случайной задачи.
3. Используйте фильтр для выбора типа задачи.
4. Добавьте новую задачу, выбрав тип и введя текст.
5. История задач сохраняется в `tasks.json`.
